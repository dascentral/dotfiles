# Interview

## Usage

```text
/resolve-copilot-comments
```

## Instructions

Review the pull request that is currently open and any **unresolved** comments that have been generated by GitHub Copilot. Make the necessary changes to address any **unresolved** comments that you feel are reasonable or necessary.

If you cannot find any open pull requests with comments, take no action.
