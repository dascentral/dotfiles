# Interview

## Usage

```text
/resolve-copilot-comments
```

## Instructions

### Step 1: Review unresolved comments

Review the pull request that is currently open and any **unresolved** comments that have been generated by GitHub Copilot. Identify comments that you feel are reasonable or necessary to address.

If you cannot find any open pull requests with comments, take no action.

### Step 2: Resolve issues

Make the necessary changes to address the **unresolved** comments that you felt were reasonable or necessary.

### Step 3: Commit changes and push

After addressing the feedback, commit all changes and push them to the remote.

### Step 4: Resolve comments

Update each **unresolved** comment and describe the fix applied or your rationale for taking no action. Resolve all conversations once complete.
