# Interview

## Usage

```text
/interview
```

## Instructions

Interview me with targeted questions to understand the feature or task I want to build. Ask one question at a time. Once you have enough context, produce a detailed implementation plan, but do not write any code or modify any files.

> Note: For full Plan Mode enforcement (read-only tools), activate Plan Mode first with `/plan`, then run this command.
